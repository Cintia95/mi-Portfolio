# Práctica Formativa Obligatoria 1 - Portafolio Personal

Este proyecto es una landing page de portafolio personal desarrollada como parte de un trabajo práctico. Incluye secciones como presentación personal, proyectos, habilidades, formulario de contacto y películas favoritas. El objetivo es demostrar el uso de HTML semántico, CSS responsivo con Flexbox/Grid, y la publicación en GitHub y Vercel.

  - **Redacta: ¿Qué ventajas encontraste al utilizar Flexbox o Grid en tu proyecto?**
